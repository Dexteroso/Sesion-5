Programa para explicar Event Listeners para los eventos y uso de bucles FOR para la ejecución iterativa

Un "listener" es un objeto o función que espera a que ocurra un evento (como un clic, un movimiento del ratón o la entrada de texto). Cuando el evento sucede, el listener lo detecta y ejecuta el código que se le ha asignado para responder.

Estructura del Event Linstener

// 1. Selecciona el elemento objetivo

const miBoton = document.getElementById('miBoton');

// 2. Define la función que se ejecutará

function mostrarMensaje() {

alert('¡Se ha hecho clic en el botón!');

}

// 3. Agrega el Event Listener

miBoton.addEventListener('click', mostrarMensaje);


En este ejemplo, miBoton es el elemento objetivo, 'click' es el tipo de evento, y manejarClick es la función que se ejecuta cuando el botón es clickeado.

Estructura del bucle For

for (inicialización; condición; incremento) {

// Código que se ejecuta en cada iteración

}
