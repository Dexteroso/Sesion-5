Ejemplos para revisar el tema de
Variables y Operadores Matemáticos

Convertidor de Metros a Kilómetros

Descripción: Un programa que convierte distancias de metros a kilómetros.





Calculadora de IMC (Índice de Masa Corporal)

Descripción: Un programa que calcula el IMC a partir del peso y la altura del usuario.







Convertidor de Temperatura

Descripción: Un programa que convierte temperaturas entre Celsius y Fahrenheit.
